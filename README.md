# job4j_grabber

## О проекте

Этот проект позволяет собирать информацию с сайтов по заданным критериям.

Функционал.

1. По умолчанию идёт сбор вакансий с сайта по ссылке
   https://career.habr.com/vacancies/java_developer.
2. Программа считывает вакансии по указанной ссылке и записывает их в базу.
3. В программе возможен парсинг других сайтов, в то числе параллельный парсинг нескольких сайтов.
4. Запуск системы по расписанию, которое указанно в настройках.
5. Доступ к интерфейсу будет через REST API
